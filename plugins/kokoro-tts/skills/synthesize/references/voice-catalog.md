## Grade Scale

| Grade | Meaning                                   |
| ----- | ----------------------------------------- |
| A     | Excellent — natural, clear, great prosody |
| A-    | Very good — minor artifacts               |
| B-    | Decent — noticeable but acceptable issues |
| C+    | Acceptable — some quality concerns        |
| D     | Poor — robotic or monotone                |
| F+    | Failing — barely usable                   |

## Recommendations

- **English default**: `af_heart` (Grade A) — best overall quality
- **Chinese default**: `zf_xiaobei` — only Chinese voice available
- **Male voice**: None recommended at this time (all grade D or below)
