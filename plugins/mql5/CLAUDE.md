# mql5 Plugin

> MQL5 development: indicator patterns, mql5.com article extraction, Python workspace, log reading.

**Hub**: [Root CLAUDE.md](../../CLAUDE.md) | **Sibling**: [quant-research CLAUDE.md](../quant-research/CLAUDE.md)

## Skills

| Skill                     | Purpose                                                      |
| ------------------------- | ------------------------------------------------------------ |
| `mql5-indicator-patterns` | Buffer management, display scaling, recalculation, debugging |
| `article-extractor`       | Extract technical trading articles from mql5.com             |
| `python-workspace`        | Configure Python workspace for MQL5-Python integration       |
| `log-reader`              | Read MetaTrader 5 log files to validate indicator execution  |

## Conventions

- Merged from `mql5-tools` + `mql5com` plugins
- MetaTrader 5 required for indicator development
- Python 3.11+ for MQL5-Python integration (`uv pip install MetaTrader5`)
